I've been building things with code for a while now, but I haven't been great about showing my work. This site is me changing that.

## Why now?

AI has changed how I build software. I can move faster, try more ideas, and actually ship things instead of letting them sit half-finished. It felt like the right time to start putting things out there.

## What to expect

This site will evolve. Right now it's simple — a homepage, some projects, and this blog. Over time I'll add more projects, write about what I'm learning, and use this as a place to experiment.

The philosophy is: ship imperfect, iterate, learn by doing.

If you're reading this, thanks for stopping by. More to come.
